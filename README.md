Get your own token and put into the config.js
Fill out the embed.json. To mention a user in the message content put {mention}.
Button URL is required to be a correctly formatted link (http/https).
Button Label is required.
Button Emoji is required. If its a default emoji leave ID blank and put it in "name".

Adjust the timeout between each message in the config.js. The time is in milliseconds.

Open a terminal in the folder, and run "npm install".
Build the project using "npm run build"
Then launch the bot with "npm start"
